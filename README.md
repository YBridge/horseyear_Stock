# 美股AI分析系统

一个基于Python和Flask的美股分析系统，提供专业级的股票分析、市场预测和投资策略管理。

## 功能特点

### 📊 单股分析
- 深度基本面分析（营收、盈利、成长、财务健康、估值）
- 技术面分析（趋势、动量、RSI、关键价位）
- 双体系综合评分（理性50% + 情绪50%）
- 交易建议（入场价、目标价、止损、仓位建议）

### 🎯 关注列表对比
- 多股票同时分析对比
- 自动评分排序
- 最佳投资机会推荐

### 📝 策略历史
- 保存投资策略
- 策略回顾和追踪

### 🤖 AI领域投资
- AI概念股专业分析
- 行业趋势判断

### 🌍 市场前瞻
- 宏观经济环境分析
- 市场情绪指标
- 理性与情绪分拆分析
- 趋势预测（1天、3天、1周）

## 安装依赖

```bash
pip install flask yfinance requests pandas
```

## 配置API密钥（可选）

系统支持OpenRouter API进行增强AI分析：

```bash
export OPENROUTER_API_KEY=your_key_here
```

如果不配置API密钥，系统将使用内置分析逻辑。

## 运行

```bash
python stock_api_server.py
```

访问 http://localhost:5000

## 页面导航

- **主页** - 单股分析
- **关注对比** - 关注列表对比分析
- **策略历史** - 查看保存的策略
- **AI投资** - AI概念股分析
- **市场前瞻** - 宏观微观分析预测

## 技术栈

- **后端**: Python + Flask
- **数据源**: Yahoo Finance (yfinance)
- **分析**: 自有双体系分析模型
- **前端**: HTML + CSS + JavaScript
